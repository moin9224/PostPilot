## 📝 Step-by-Step Usage

### **Step 1: Get a Gemini API Key**
1. Go to https://aistudio.google.com/app/apikey
2. Click "Get API Key"
3. Create a new key
4. Copy it

### **Step 2: Add Gemini Key to Settings**
1. Go to Settings → API Keys
2. Paste your Gemini key
3. Click "Add"

### **Step 3: Convert Any URL**
1. Find an article you like
2. Copy the URL
3. Go to Content Generator
4. Paste URL into the input
5. Click "Convert"
6. Wait ~5 seconds for AI to process
7. Get your post!

### **Step 4: Use Your Post**
- Copy the main post
- Copy any hashtags
- Copy variations you like
- Paste into LinkedIn
- Done! ✅

## 🔧 How It Works Under the Hood

```
URL Input
    ↓
Backend fetches page content
    ↓
Extracts main text (removes HTML)
    ↓
Sends to Gemini AI
    ↓
AI generates LinkedIn post
    ↓
AI generates 2 variations
    ↓
AI suggests 5-7 hashtags
    ↓
Returns all 3 to frontend
    ↓
You copy and use!
```

## ⚙️ Technical Details

### **API Endpoint**
```
POST /api/url-to-post
Body: { "url": "https://example.com/article" }
Response: {
  "content": "Main LinkedIn post...",
  "variations": ["Variation 1...", "Variation 2..."],
  "hashtags": ["#hashtag1", "#hashtag2", ...],
  "estimatedReach": 1200
}
```

### **Error Handling**
- Invalid URL → Clear error message
- Can't fetch page → Tells you why
- No content found → Suggests longer articles
- AI fails → Suggests adding Gemini key

## 🚨 Troubleshooting

### **Issue: "Cannot fetch URL"**
- **Solution:** Check URL is correct
- **Solution:** Try a different article
- **Solution:** Make sure page loads in browser

### **Issue: "AI generation failed"**
- **Solution:** Make sure Gemini API key is added in Settings
- **Solution:** Make sure key is valid (not expired)
- **Solution:** Try a different URL with more content

### **Issue: "Could not extract content"**
- **Solution:** URL has no text content
- **Solution:** Try an article with 500+ words
- **Solution:** Avoid video-only pages

### **Issue: Post quality not great**
- **Solution:** Use articles with clear insights
- **Solution:** Use articles with data/stats
- **Solution:** Try a more interesting article
